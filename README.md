# Java Learning Journey

Este repositório documenta minha jornada de aprendizado em Java, desde os conseitos básicos até tópicos avançados do desenvolvimento back-end.
Objetivos:
- Consolidar conhecimento em java.
- Praticar lógico de programação.
- Aprender Programaão Orientada a Objetos (POO)
- Evoluir para o desenvolvimento Back-end com Spring Boot.

Tecnologias que serão estudas:
-Java
-POO
-Collections
-JDBC
-MySQL
-Spring Boot
-APIs REST

Este repositório será atualizado constantemente conforme avanço nos estudos.
